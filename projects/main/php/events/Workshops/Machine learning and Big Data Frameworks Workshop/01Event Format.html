Event Format

 

 

It will be a 6 hours workshop on a single day. The first two hours of the workshop in topics 1, 2 shall cover the architecture ideas for data mining. The remaining workshop in topics 3,4 shall cover data science case studies with popular data mining models. 

 

1. Machine Learning Systems : 
1.a. Knowledge Discovery in Databases : 
Cross Industry Standard Process for Data Mining 
Predictive Model Markup Language
Complex Event Processing 
Real-time business intelligence
1.b Lambda Architecture : 
Concept Maps and Data Pipelines in Functional programming
Industrial Internet and Semantic Web
Computational and Machine Intelligence

2. Big Data Frameworks :
Data Representation : 
Learning paradigms
Analytics stacks
Distributed processing
Data products experimentation

3. Data Science Cases :
Data Visualization
Data Munging
Data Sampling 
Hypothesis Testing 
Goodness of Fit
Nonparametric statistics, Descriptive Statistics, Inferential Statistics

4. Data Mining Models : 
4.a Feature Selection : Dimensionality Reduction, Feature Extraction, 
4.b Clustering : Hierarchical Clustering, Nearest Neighbours, 
4.c Classification : Binary Classification, Naive Bayes, Decision Trees, Logistic Regression, Boosting,
4.d Probabilistic Rules : Frequent Patterns, Expectation Maximization, Graphical Models, Network Analysis
4.e Regression : Neural Networks, Kernel Machines, 
4.f Information filtering : Collaborative Filtering, Novelty Detection
