Machine Learning and Big Data Frameworks      Workshop

 - Brought to you by Shaastra 2015 in association with Flutura

 

 

 

Machine Learning and Big Data Frameworks workshop is about reviewing machine learning, data mining and big data frameworks.The pedagogical approach is to explain the intuition for an algorithm or a technique by demonstrating an easy to understand example.

 

  By the end of the workshops, participants shall
- Become familiar with the big data technology landscape
- Learn step-by-step algorithmic detail for mapping learning models to implementation stacks
- Rapidly try out new ideas and test hypotheses
- Search and interpret open courseware on data analytics
