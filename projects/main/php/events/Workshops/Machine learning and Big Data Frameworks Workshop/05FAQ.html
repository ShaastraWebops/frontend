Frequently  Asked Questions

 

Q) Is there any prerequisite needed for the workshop?

A) Participants are expected to have basic hold on atleast one programming language. The programming languages for data mining include SQL, R, Python, Java, Scala, Julia.

 

Q) Do I need to bring any tools for the workshop?

A) Yes, participants are required to bring their own charged laptops to the workshop. They should have Java, Python and R installed on their laptops.

 

Q) Can we register on-the-spot for this workshop?

A) No. It is a pre-registered workshop. The registrations have started.

 

Q) What documents I need to carry for the workshop?

A) Photo ID Card from a recognized institution/government organization.
