Basics

  1. How did I come here %-/ ?

To come here, you need to add the 'edit' parameter to the url. Copy the url and add '&edit' to the end of any event page, and you can make the page editable.

  2. That's great, but how do I edit content ?

Editing content is simple ! Once in the 'Edit Mode' click anywhere in this area and begin typing ! You'll see a style-bar on top which will help you format your code (Most basic functionality is there). Bold, Underline, Italics, Horizontal Line, Table, Alignment, Add image .... customize to your hearts content. After editing, click the 'Save' button in the style-bar (It's the first button - next to Bold)

  3. Woah, that's pretty neat ! How do I change the title of the tab ?

Changing title is same as content, click on the tab's title on top and a popup will appear where you can change it.

  4. Wait, I don't have any tabs right now, how do I add a new one ?

The '+' on the top will help you out.

Content Styling

You guys did a pretty neat job, but it doesn't look the way I want it to !

  1. The color (OR) size (OR) font is different. I copied this from GMail (OR) Word ... It just refuses to look the way I want it to !

There are many reasons for this. Mainly due to some things that Microsoft does which break everything else ( Try Linux :P ). But there are some fixes for it.

  • Try using the 'Remove Format' button on top - The T with the x on the side.
  • Try using the 'Paste as Text', 'Paste from Word' options. They should most probably help you out.

  2. How do I add an image ?

You can add an image using the 'Image' button on the style-bar. This asks for a URL - So, the image must be uploaded somewhere before. Then you can edit the size etc. Try clicking an Image after it is added, you can resize and move it.

  3. How do I add a link to an image ? (Sheesh, Spons people are very painful)

I need to add links to images (Yes, I've had spons junta calling me up everyday for the same last year ... Sigh). Well, it's pretty easy. Right click the image and go to image-properties. In the popup go to 'Link' tab and give the URL. Also, most probably you want to open it in a new tab, so select that option inn the dropdown (The _blank)

If you have an image on your computer, upload it to a safe location in google drive or other uploading site and then use it.

  4. If I type a link, it doesnt get automatically linked ! What do I do ?

This one is very similar to Word/GMail - Use the 'Link' button in the style-bar. Again to open in a new tab use the Second tab's 'Target'
