Arduino Programmer's Challenge

 

Arduino is a very common electronics prototyping platform. Arduino coding is so easy to learn that somebody with decent experience with C or C++ can pick it up within an hour. There are coding challenges for C and python so why don’t we have one such competition for Arduino. In this challenge, the participant will be asked to solve a logical hands-on puzzle with the Arduino or write a cryptic code for the Arduino.
