Gear up for the App Challenge – the Technical know-how

Various companies provide Electronic Health Records (EHR) solutions to Hospitals for maintaining patient data in a safe and secure way.  FHIR is a specification which strives for interoperability between various  EHR providers so that data stored in the EHR's can be viewed and manipulated.  This would open up a world of opportunities for various developers to build applications (web/mobile/desktop).
