Frequently Asked Questions

 

1. What will be the size of the team?

  • The maximum size of the team is restricted to 3 members.

 

2. Do all members need to be from the same college?

  • No, team members can be from different colleges.

 

3. Should the registration for the event be done online?

  • No, the registration for this event should be done on the spot.

 

4. Do we need to have any prerequisite knowledge on forensics?

  • NO. Neither the preliminary nor the finals will be technical.
