There. Will. Be. Chaos 

 

It is an individual event. The registration is online. Anyone with a Shaastra ID or IIT-M login can register. 

There are no prelims, no finals. Got the answer? Submit it and walk away with a reward.

The event will commence on January 3 (0900hrs) and will close on Jan 7 (0000hrs).

Problems will be released at random intervals of time throughout the course of the event.

The questions will be wide ranging. Most of the problems would be puzzles, ciphers, and coding questions( problems whose solution must be in code ).

However. There. Will. Be. More Chaos.

The remaining questions will be a surprise. The kind of ones which would make you think the chances of you solving them is less than that of the survival of a character in the Game of Thrones.

There will be cleverly concealed clues. Look out.

Submit the solution in .txt format to the link provided.

The first person to crack the right answer will be sent a URC (Unique Redemption Code) via SMS. The code can be redeemed at the Hospitality Desk for a reward.

You can win as many as 10 codes per day per participant.

You can redeem each code individually for a single reward or combine them to get a bigger reward. All the codes must be redeemed on the day they are issued. They become invalid after midnight.
 

*There is a small workaround by which you can be notified when a problem is released if you want to save yourselves the pain of checking the web page every minute. Good luck figuring it out.
