Project X
Timing: 11:30AM - 1:00 PM, 21st September

Project X is one of the colossal events at Shaastra in terms of the participation witnessed. The event is all about solving intriguing problems using basic day-to-day materials. Since the only requirements for the event are creativity, enthusiasm and craftsmanship, the event has been a big crowd puller with more than 1000 people participating each year.

On-spot registrations in teams of 3. Prizes worth Rs 3,000 up for grabs.
